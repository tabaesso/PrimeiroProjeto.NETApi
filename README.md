## Basic Setup

1. [Install Node.js](https://nodejs.org/)
1. [Install .NET Core 2.2](https://dotnet.microsoft.com/download/)
2. Fork the [ProAgil](https://github.com/vsandrade/ProAgil/fork)
3. Clone the repo you just forked.
